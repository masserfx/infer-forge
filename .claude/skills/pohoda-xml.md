# Pohoda XML API
## Kódování: Windows-1250 (NE UTF-8!)
## Wrapper: dat:dataPack version="2.0", ico="04856562", application="INFER_FORGE"
## Typy: nabídka (ofr:offer), objednávka (ord:order), faktura (inv:invoice), adresář (adb:addressbook)
## Validuj XML proti XSD před odesláním
## Datum: YYYY-MM-DD, čísla dokladů: unikátní prefix
## Běžné chyby: duplicitní číslo, timeout, neplatné IČO
